# 3. Generator 函数

- 生成器函数:
    1. 函数中可以使用 yield 关键字，会暂停函数的执行
    2. 生成器函数调用后，不会立即执行，而是返回一个迭代器对象
    3. 迭代器对象包含 next 方法，调用 next 方法会执行函数体，直到遇到下一个 yield 关键字，next接收参数作为上一个 yield 表达式的返回值
    4. value 属性表示当前 yield 表达式的值
    5. done 属性表示函数是否执行完毕
