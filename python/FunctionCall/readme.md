# 幻觉

- LLM 以为自己听懂了你的问题，并给出来一堆看似正确的答案，但实际上这些答案是错误的。

- 大模型的训练方式是通过学习语言的结构和模式，使其生成自然语言相似的文本

- 借助 FunctionCall 这一机制，给大模型提供一些工具函数，大模型根据需要自行选择合适的工具函数
    - 比如：

      用户：我附近有哪几家瑞幸咖啡
      LLM：理解用户的问题，然后调用高德地图的相关接口，获取用户当前位置附近的瑞幸咖啡店信息。
      根据获取到的信息，生成自然语言的回答
