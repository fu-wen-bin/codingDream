# 如何让 LLM 接入第三方平台

- AIGC (大模型 == 你问他答，数据都是曾经投喂过的)
- AGI  (人工智能，拥有自我思考的能力)
- 目前的 LLM 都是静态的，无法实时获取数据

- 明天江西省南昌市明天的天气情况  (实时性问题目前大模型无法解决)

1. function calling
    - 通过函数调用的方式，获取实时数据
    - 需要第三方平台提供 API 接口
    - 需要 LLM 支持函数调用
    - 工具函数的标准混乱
2. MCP协议
    - MCP 协议是一个标准化的函数调用协议
    - 通过 MCP 协议，可以让 LLM 接入第三方平台
    - MCP 协议可以解决函数调用的标准混乱问题
    - MCP 协议可以让 LLM 获取实时数据
